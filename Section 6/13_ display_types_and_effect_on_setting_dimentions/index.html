Display:Block = Block elements always start on a new line and take up the full available width. They respect Width, Height, and all Margin/Padding properties.
Display:Inline = Inline elements flow horizontally on the same line as text. They IGNORE Width, Height, Margin-Top, and Margin-Bottom.
Display:Inline-block = The perfect hybrid. It respects block properties but flows inline. They respect Width, Height, and all Margin/Padding properties, AND stay on the same line.
