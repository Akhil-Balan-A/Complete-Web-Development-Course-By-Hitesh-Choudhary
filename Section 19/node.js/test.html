1. Expandable FAQ Section using <details>

What is HTML's <details> tag?

The <details> HTML element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state. A <summary> element is used as the visible heading.

Can I use this for menus?

Yes, absolutely! It is great for creating hierarchical menus, nested lists, or any component where you need to hide and show a block of content.

Does it require any JavaScript?

No! This feature is natively supported by modern browsers and requires zero JavaScript code to handle the opening and closing behavior.
